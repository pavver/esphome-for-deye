1. Overview

This protocol is used for communication between the monitoring of the energy storage inverter and DSP. It adopts the MODBUS RTU communication protocol. This protocol can read the operating information of the inverter in real time and control the inverter.

2. Physical Interface

Uses RS485/RS232, asynchronous transceiver mode, master-slave mode, fixed baud rate.

  • Baud rate: 9600bps
  • Parity: None
  • Data bits: 8
  • Stop bits: 1

Frame interval time requirement

3. Data Frame Format

Slave Address Function code Data CRC Check
8-Bits 8-Bits Nx8-Bits 16-Bits

Slave Address field: is the corresponding slave address, which must match the slave address of the inverter.

Function code field: function code, currently only 03H and 10H function codes are open.

Function code(Hex) Description Register Address Functionality
02H Read Discrete Inputs Read Fault Information Register Content
03H Read Holding Registers 0~59/500~2000 Read Settings Register Content
04H Read Input Registers Read Inverter Information Content
05H Write Single Coil Enable/Disable Machine Settings Function
06H Write Single Holding Register Set Single Register Function
10H Write Multiple Holding Registers 60-499 Set Multiple Registers Function

Data field: includes starting register address, data length, number of data bytes, data content. All are high byte first, low byte last.

CRC Check field: CRC lookup table checksum method, low byte first, high byte last.

4. Error Message and Data Handling

Slave reply (16-bit):

Slave Address Function code Error code CRC Check
xx xx|0x80 xx Low byte High byte
xx xx

When the inverter communication module detects errors other than CRC checksum errors, it must reply to the master with information. The highest bit of the function code is set to 1, which means adding 128 to the function code sent by the master.

Error codes responded by the inverter communication module:

0x01 Illegal function code. The server does not understand the function code.

0x02 Illegal data address. (requested)

0x03 Illegal data value. (requested)

0x04 Server failure. The inverter communication module cannot output data fault during execution.

5. Detailed Protocol Description

0-59 register addresses are readable register types, function code 0x03.

60-499 register addresses are readable and writable register types, function code 0x10.

500-2000 register addresses are readable register types, function code 0x03.

F24 DC insulation impedance failure / Array insulation impedance fault PV isolation resistance is too low.
1. Check that the connection of PV panels and inverter is firm and correct.
2. Check whether the PE cable of the inverter is connected to ground.
3. Seek help from us if it cannot go back to a normal state.
F26 The DC busbar is unbalanced / DC bus unbalanced 1. Please wait for a while and check whether it is normal.
2. If still the same, turn off the DC switch and AC switch, wait for one minute, and then turn on the DC/AC switch.
3. Seek help from us if it cannot go back to a normal state.

1. Check the parallel setting according to the instructions.
2. Check the connection of the CANBus.
3. Seek help from us.

1. Please confirm if the grid is lost or not.
2. Check if the grid connection is good or not.
3. Check if the switch between the inverter and grid is on or not.
4. Seek help from us if it cannot go back to a normal state.
F41 Parallel system Stop / Parallel system shutdown fault In a parallel system, due to other inverter faults.
1. Wait for minutes then check all inverters in this parallel system.
2. If the inverter can’t go back to a normal state, record fault codes of all inverters, then seek help from us.

1. Check if the AC voltage is in the range of the standard voltage in the specification.

3. Seek help from us if it cannot go back to a normal state.

3. If the inverter can’t go back to normal after load reduction, seek help from us.
